# Analizador de Temperaturas

Trabajo práctico realizado para la Lección 6 – Sentencias Iterativas (Python).

El ejercicio fue desarrollado siguiendo la guía del profesor y con apoyo
explicativo para comprender el uso de bucles, funciones y validaciones.

Objetivos:
- Uso de for y while
- Centinelas
- Separación de lógica en funciones
- Cálculo de estadísticas sin funciones built-in
